# 스토리시드 자동 업데이트 설정 가이드 (Tauri v2 · GitHub Releases · 무료)

이 프로젝트는 **Tauri v2**입니다. (`src-tauri/Cargo.toml`의 `tauri = "2"`, `tauri.conf.json`의 `"$schema": ".../config/2"`로 확인)

자동 업데이트는 **완전히 무료**로 운영됩니다. 배포처는 GitHub Releases(무료), 업데이트 서명은 Tauri가 무료로 만들어주는 키를 씁니다. (윈도우 SmartScreen / 맥 Gatekeeper 경고를 없애는 'OS 코드서명'은 유료지만, 자동 업데이트 동작과는 무관합니다.)

이미 코드는 배선해 두었어요. 아래 **3가지 값**만 채우면 작동합니다.

---

## 1단계: 업데이트 서명 키 만들기 (1회, 무료)

내 컴퓨터 터미널에서:

```
npm run tauri signer generate -- -w ~/.tauri/storyseed.key
```

- 비밀번호를 물어보면 원하는 걸 정해서 입력(빈 값도 가능).
- 끝나면 **공개키(public key)**가 화면에 출력되고, 개인키는 `~/.tauri/storyseed.key`에 저장됩니다.

## 2단계: 공개키를 conf에 넣기

`src-tauri/tauri.conf.json`을 열어 두 곳을 바꿉니다.

```json
"plugins": {
  "updater": {
    "endpoints": [
      "https://github.com/내깃허브아이디/저장소이름/releases/latest/download/latest.json"
    ],
    "pubkey": "1단계에서 나온 공개키 붙여넣기"
  }
}
```

- `내깃허브아이디/저장소이름`을 실제 값으로 교체.
- `pubkey`에 1단계 공개키를 그대로 붙여넣기.

## 3단계: GitHub 저장소에 비밀키 등록

저장소 → **Settings → Secrets and variables → Actions → New repository secret** 에서 2개 등록:

| 이름 | 값 |
|---|---|
| `TAURI_SIGNING_PRIVATE_KEY` | `~/.tauri/storyseed.key` 파일의 **내용 전체** |
| `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` | 1단계에서 정한 비밀번호 (없으면 빈 값) |

> 개인키 파일 내용은 `cat ~/.tauri/storyseed.key`로 확인해 통째로 복사하세요.

## 새 버전 배포하는 법 (평소 루틴)

1. 코드를 고친 뒤 **버전을 올립니다.** 세 곳을 같은 번호로:
   - `package.json`의 `"version"`
   - `src-tauri/tauri.conf.json`의 `"version"`
   - `src-tauri/Cargo.toml`의 `version`
   - (표시용) `src/index.html`의 `APP_VERSION`
2. 태그를 만들어 push:
   ```
   git add -A && git commit -m "v2.0.1"
   git tag v2.0.1
   git push && git push origin v2.0.1
   ```
3. GitHub **Actions → 스토리시드 릴리스 (자동 업데이트)** 가 자동 실행됩니다.
   - mac/windows 설치본과 함께 **`latest.json`(서명 포함)**이 Release에 올라갑니다.
4. 끝! 이미 스토리시드를 쓰는 사용자는 앱을 켜면 몇 초 뒤 "새 버전이 있어요" 안내를 받고,
   확인을 누르면 자동으로 내려받아 설치·재시작됩니다.
   (환경설정 → 정보 → **지금 확인**으로 수동 확인도 가능)

---

## 작동 원리 한눈에

```
새 태그 push
   → GitHub Actions가 빌드 + 서명 + latest.json 생성
   → GitHub Releases(무료)에 업로드
사용자 앱 실행
   → tauri.conf의 endpoints(latest.json) 확인
   → 현재 버전보다 높으면 다운로드 → 서명 검증 → 설치 → 재시작
```

## 자주 묻는 것

- **정말 무료인가요?** 네. GitHub Releases 저장·전송 무료, Tauri 업데이터·서명 무료입니다.
- **패치 파일만 배포할 수 있나요?** Tauri 업데이터는 전체 설치본을 내려받습니다(델타 아님).
  스토리시드는 설치본이 가벼워(수 MB) 실사용엔 문제없어요.
- **OS 코드서명(유료)은 꼭 필요한가요?** 아니요. 없어도 자동 업데이트는 작동합니다.
  다만 설치 시 "확인되지 않은 개발자" 경고를 한 번 넘겨야 합니다.
- **맥/윈도우 둘 다 되나요?** 네. 워크플로가 양쪽 설치본과 각 플랫폼용 서명을 함께 올립니다.
